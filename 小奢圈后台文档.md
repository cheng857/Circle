# 小奢圈后台文档

## 用户管理

### 用户列表

1. 搜索：圈ID，昵称，来源渠道
2. 筛选：性别，城市，在线状态，身份信息
3. 个人资料：
   - 基本资料：圈ID，在线状态，照片集，昵称，性别
   - 详细资料：生日，城市，职业，身高，体重，个性签名，个性标签，社交账号
   - 定位信息：最后一次获取到的经纬度相应的城市
4. 身份信息：
   - VIP：开通则展示有效时间
   - 真人认证：可查看认证照片
   - 人气认证：可查看自我介绍视频，并展示认证得分
5. 账号信息：
   - 注册账号：手机号码
   - 注册时间：YYYY-MM-DD hh:mm:ss
   - 来源渠道：渠道ID
   - 钱包余额：钻石数
   - 绑定关系：上级用户ID，下级用户ID
   - 账号状态：正常/冻结至:YYYY-MM-DD hh:mm:ss
6. 操作：
   - 编辑资料：可将用户填写的个人资料删除，无头像、昵称则展示默认，并向用户发送系统通知（后续所有设计删除用户资料、驳回审核、解除认证的功能，都需要选择理由或自行填写理由，选择列项内容根据场景确定）
   - 换绑手机：修改绑定手机号
   - 充值：钻石充值
   - 流水查询：时间，金额（人命币/钻石），进/出账，关联ID，流水说明
   - 冻结：设置冻结时间

## 认证审核

### 真人认证

1. 搜索：圈ID
2. 认证信息：圈ID，认证时间，上传的照片集封面，人脸比对图片
3. 认证状态：
   - 未通过：操作-通过
   - 已认证：操作-解除
   - 平台解除

### 人气认证

1. 搜索：圈ID
2. 认证信息：圈ID，认证时间，自我介绍视频，历史人脸比对图片
3. 认证状态：
   - 视频审核中：操作-通过，打回
   - 视频审核未通过：操作-通过
   - 社区投票中：操作-打回
   - 社区投票未达标
   - 平台审核中：操作-通过，打回
   - 平台打回
   - 已认证- X%：操作-解除
   - 平台解除

### 动态发布审核

1. 搜索：圈ID
2. 动态信息：圈ID，发布时间，动态内容（话题，文本，图片/视频），位置信息
3. 话题审核：
   - 如果动态中创建了新话题，则展示话题审核
   - 审核内容：话题封面，话题介绍
4. 发布状态：
   - 审核中：操作-通过，打回
   - 审核通过：操作-打回
   - 自动通过：操作-打回
   - 平台打回

### 照片集上传审核

1. 搜索：圈ID
2. 动态信息：圈ID，上传时间，上传的照片，历史人脸比对图片
3. 发布状态：
   - 未通过：操作-通过
   - 已通过：操作-下架
   - 平台下架

### 举报审核

1. 搜索：举报人圈ID，被举报人圈ID
2. 举报信息：
   - 举报人圈ID，被举报人圈ID，举报时间
   - 举报场景：个人主页，会话页，发布动态，动态评论（可查看相应信息）
   - 描述：文本，图片/视频，选择的聊天记录
3. 审核状态：
   - 审核中：冻结，无需处理
   - 冻结中：恢复
   - 已恢复：-
   - 无需处理：冻结

## 平台互动消息

### 聊天消息

1. 以用户维度，按最新消息时间列表展示
2. 点击可查看该用户的聊天列表，以及对应的聊天记录
3. 操作：可手动屏蔽指定聊天记录

### 圈子广场

1. 根据发布时间由新至旧展示圈子广场的所有动态
2. 可查看：动态内容，评论内容，点赞用户，浏览用户
3. 操作：打回动态，删除评论

### 话题广场

1. 根据广场排序展示所有话题
2. 可查看：话题介绍，话题封面，创建用户圈ID，浏览量，排序，话题内动态
3. 操作：删除话题，排序，编辑话题（修改介绍，上传封面，转移创建者）

## 系统配置

### 平台身份权益配置

以身份维度划分权益：普通男用户，普通女用户，真人认证男用户，真人认证女用户，人气认证用户，白银会员，黄金会员

| 权益类型     | 数据类型 | 说明          |
| ------------ | -------- | ------------- |
| 解锁圈聊     | Int      | 每日可解锁X人 |
| 发布动态     | Int      | 每日可发布X条 |
| 解锁评论     | Int      | 每日可发布X条 |
| 专属勋章     | Bool     | 是否支持      |
| 优先推荐     | Bool     | 是否支持      |
| 人气板块     | Bool     | 是否支持      |
| 红包照片     | Bool     | 是否支持      |
| 高级筛选     | Bool     | 是否支持      |
| 专属客服     | Bool     | 是否支持      |
| 解锁关注限制 | Int      | 每日可关注X人 |
| 解锁微信     | Int      | 每日可解锁X人 |
| 访客查看     | Bool     | 是否支持      |
| 首页推荐隐藏 | Bool     | 是否支持      |
| 隐藏地理位置 | Bool     | 是否支持      |
| 隐身模式     | Bool     | 是否支持      |
| 足迹隐藏     | Bool     | 是否支持      |
| 身份保密     | Bool     | 是否支持      |

### 系统通知配置

配置字段：通知文本，场景，发送时间

### 会员开通配置

1. 选择会员等级：白银会员，黄金会员
2. 选择开通方式：直充，续费
3. 配置开通周期：周期名-X个月- X天
4. 配置金额/货币：钻石- X，三方支付- ¥X

### 礼物库配置

配置字段：礼物图片/特效，礼物名称，价格，序号

### 钻石充值配置

1. 充值比例配置：¥1=X钻石
2. 钻石商品表配置：钻石数量，排序，优惠金额（不填写则按照充值比例换算）

### 红包照片配置

配置字段：最多设置红包照片数（小于等于5），可选择金额

### 敏感词库配置

配置字段：敏感词内容，匹配规则（精准匹配/完整包含）

### 昵称库配置

分两段配置昵称，配置后，从两段昵称库中随机组合

### 职业库配置

分两级配置职业，可排序

### 个性标签库配置

配置字段：标签内容，Icon

### 占位虚拟用户库配置

配置字段：性别，头像上传，昵称

### 人气投票配置

配置字段：每日最多可投票用户数

## 客服IM后台

基础的Web端客服会话系统，实现会话列表、文本/图片/视频发送、聊天记录同步功能即可

## 财务管理

1. 展示字段：订单ID，用户圈ID，商品名，充值金额，创建订单时间，支付时间，订单状态（待支付、已支付），退款状态
2. 已支付订单可修改退款状态：申请退款，待退款，已退款

## **数据统计**

### 埋点文档

埋点：设备类型、ID、IP、模块、位置、时间、动作

### 数据统计

1. 注册数：当日创建账号的用户（绑定手机即创建）
2. 注册激活数：当日注册后进入首页的用户
3. 活跃数：当日在APP内停留10分钟的用户
4. 注册活跃数：当日注册后在平台内停留10分钟的用户
5. 次日留存数：昨天注册后，当日仍进入首页的用户
6. 三日留存数：两天前注册后，当日仍进入首页的用户
7. 七日留存数：六天前注册后，当日仍进入首页的用户
8. 充值数：当日充值钻石/会员的用户
9. 注册充值数：当日注册后充值钻石/会员的用户

### 筛选：

1. 设备类型：安卓，iOS
2. 城市
3. 日期范围：当日用户，3日总用户，7日总用户
